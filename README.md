Ext4-ux-form-XCheckbox
======================

Ext.ux.form.XCheckbox - checkbox with configurable submit values - plugin for ExtJS 4

# Installation

Place the folder **ux** in your project.

Do not forget to use Ext.Loader to be able to load the file.
